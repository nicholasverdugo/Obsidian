# Implementation of a Linear Regression Model
- #Linear-Regression implemented in code
- Dataset: {(xi, ti)}^n, i=1
- Regression task 
	- ti E R (labels)
	- xi E R (inputs)
- Feature Space: 
	- phi : R -> R^M
	- x->  (x^1, x^2, x^3, xxx, x^m)
	- m = hyperparameter
- Mapper function (or model):
	- f(phi(x), w)=w0 + w1 * x^1 + w2 * x^2 + ... + wm * x^m
	- SUM(wj)
